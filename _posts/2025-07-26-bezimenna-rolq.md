<div class="poem3">

Колко дръпки дим <br/>
ми трябват, за да се осмеля? <br/>
Милея ли въобще <br/>
за себе си? <br/>
 <br/>
Служа на компромис - <br/>
неизгоден, <br/>
покварен от пристрастия. <br/>
 <br/>
Обгарям се. <br/>
Не ме пуска. <br/>
А защо го допускам? <br/>
 <br/>
Стремеж към страданието. <br/>
Роб на болката. <br/>
Сам се срязах -  <br/>
преди да ме отрежат те. <br/>
