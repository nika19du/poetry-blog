<div class="poem3">

Под завивките с нея <br/>
светът е друг. <br/>
Отвъд външното -<br/>
навътре във вътрешното.<br/>
Любовен еликсир – <br/>
цяр за зависимите субстанции. <br/>
<br/>
Оплитаме дъх на спомени – <br/>
мека топлина, <br/>
която преобразява същността. <br/>
 <br/>
Кой кого води - <br/>
въпрос на его. <br/>
Кой с кого е - <br/>
егото говори по-силно<br/>
от самия теб. <br/>
<br/>
Ала с нея <br/>
то премлъква, <br/>
свива се в тишина, <br/>
за да съхрани <br/>
вас самите. <br/>
