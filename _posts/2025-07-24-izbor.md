<div class="poem3">

Зажаднял за живот. <br/>
Танцувай. Пирувай! <br/>
Колко пъти още <br/>
ще ти бъде даден шансът? <br/>
 <br/>
Избери се. <br/>
Наложи се. <br/>
 <br/>
Единствено <br/>
своята отговорност носи. <br/>
 <br/>
И стой тежко <br/>
зад намеренията си. <br/>

</div>
