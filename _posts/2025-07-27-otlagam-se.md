<div class="poem3">

Прелом без милост <br/>
тлее още в мен. <br/>
Ти продължи смело, <br/>
а аз <br/>
осмислено се отлагам. <br/>
 <br/>
Губещ ход <br/>
в таен тест. <br/>
 <br/>
Кривнах порочно, <br/>
по поръчка - <br/>
за измислица <br/>
не точна. <br/>
 <br/>
Отсреща е само <br/>
спретнато <br/>
мълчание. <br/>
Нямаше край
