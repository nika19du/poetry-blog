Надеждата е условна при свободната воля. <br/>
Можеш ли да се надяваш без привързаност? <br/>
Или надеждата винаги е вид контрол, маскиран като доверие? <br/>
 <br/>
Колкото повече кроя надежда към теб, <br/>
толкова повече тя се разпада, защото ти можеш да избереш друго. <br/>
 <br/>
Колкото повече проектираш надежда върху нещо или някого със свободна воля, <br/>
толкова по-условна и крехка става тя. <br/>
 <br/>
А надеждата се основава на очакване. <br/>
А очакването на нужда. <br/>
А нуждата на страх. <br/>
 <br/>
Затова, когато насочиш надежда към нещо, което не можеш да контролираш: <br/>
човек, съдба, избор,  <br/>
тя вече не е сигурна, не е „чиста“. <br/>
Тя зависи. <br/>
А щом зависи - става условна. <br/>
 <br/>
И накрая ограбваш себе си в надежди. <br/>
