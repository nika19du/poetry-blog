<div class="poem3">

Няма как <br/>
да се повтори <br/>
нещо незабравимо. <br/>
В това е чарът <br/>
на уникалността. <br/>
 <br/>
Като смеещ се сезон, <br/>
сменящ цвета си <br/>
щом бегло го следиш. <br/>
 <br/>
Веднъж изпитано - <br/>
трудно <br/>
възвръщаемо. <br/>
 <br/>
Ако го насилстваш <br/>
повторно - <br/>
ще се <br/>
изврати.
