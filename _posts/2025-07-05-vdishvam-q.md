<div class="poem2">
Вдишвам я дълбоко, <br/>
като последна цигара преди безсъние. <br/>
Затъвам <br/>
в бездна възмездна. <br/>
Разлютяват ме <br/>
горчивите викове на миналото. <br/>
Стържа крехкостта си <br/>
до самозабрава. <br/>
Загубих се. <br/>
Уязвимостта се разголи - <br/>
достатъчно, <br/>
за да срути всичко вътре. <br/>
</div>
