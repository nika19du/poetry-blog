Толкова  <br/>
несъществени  <br/>
неща има.  <br/>
  <br/>
Блазнят те  <br/>
за пропилян блян.  <br/>
  <br/>
Подтикват -   <br/>
постоянно промиване  <br/>
после: осмиване.  <br/>
  <br/>
Раздерателно  <br/>
ръсят  <br/>
клишета:  <br/>
„няма винаги да е все така“...  <br/>
(за кого?)  <br/>
за тях, сигурно.  <br/>
  <br/>
Как да ме вълнува,  <br/>
като не ме определя.  <br/>
  <br/>
Извратени  <br/>
Известители.  <br/>
  <br/>
Остана ми да пропиша.  <br/>
Без да подозирам  <br/>
какво ще излезе.  <br/>
  <br/>
Не знам кога ще спра  <br/>
поезията.  <br/>
(Може би преди  <br/>
себе си.)  <br/>
