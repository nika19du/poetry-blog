<div class="poem3">

Отпусни се. <br/>
Временно притъпи пристъпа. <br/>
Премини — латентно. <br/>
Да го изкорениш като беглец. <br/>
Какъв страхливец? <br/>
 <br/>
Измени се в чужда угода — <br/>
като измет. <br/>
 <br/>
Вменяем
за трохи внимание. <br/>
